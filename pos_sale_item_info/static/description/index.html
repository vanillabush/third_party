Odoo Point of Sale (POS) with On Hand Quantity

Show Product Stock Quantity on Point of Sale

This module show Product Stock Quantity on Point of Sales.


Main Features:

  • This module will allow you to show Product Stock Quanity (Quantity on hand) on POS.
  • If Show Available Qty in POS? is set on Product Variant then you can show that product quanity.
  • In this module, has only stockable product has show the 'Show Available Qty in POS?'.
  • For more details see video and below screenshots.

Edition Compatibility:
